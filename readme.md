# 常用git命令清单
## git 名词解释
* workspace: 工作区
* stage：暂存区
* repository：仓库
* remote：远程仓库
## 新建代码库
\# 在当前目录初始化git库
`$ git init`
\# 新建一个目录，将其初始化为git代码库
`$ git init [project-name]`
\# 下载一个项目和它的所有历史版本
`$ git clone [url]`

* git init 初始化git仓库
* git add filename 添加filename文件到git仓库
* git commit -m "some words" 提交更改，“some words”为本次commit 的信息。
